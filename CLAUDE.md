# Roblox — CLAUDE.md

> Este vault está potenciado por el plugin [Bedrock](https://github.com/iurykrieger/claude-bedrock).
> Las instrucciones a nivel de plugin (tipos de entidad, reglas de escritura, tags, flujo de git) se cargan automáticamente.
> Este archivo describe lo específico de ESTE vault.

## Propósito

Vault de gestión de conocimiento personal (segundo cerebro) para aprendizaje, proyectos, ideas y organización de la vida. Pensado para centralizar notas sobre desarrollo en Roblox/Luau, aprendizaje, proyectos personales y cualquier conocimiento que quieras conectar y recuperar.

## Idioma

Todo el contenido de este vault se escribe en **Español (es)**.
Al crear o actualizar entidades, usa español para todo el contenido textual.
Las claves del frontmatter se mantienen en inglés. Los términos técnicos en inglés son aceptables.

## Dominios

Este vault utiliza los siguientes domain tags:

- domain/learning
- domain/career
- domain/projects
- domain/ideas
- domain/health
- domain/finance

Al crear entidades, usa tags `domain/<nombre>` de esta lista.
Se pueden añadir nuevos dominios a medida que el vault crezca.

## Referencia rápida

| Acción | Skill |
|---|---|
| Buscar y consultar el vault | `/bedrock:ask` |
| Ingestar fuentes externas (Confluence, Google Docs, repositorios de GitHub, URLs remotas y cualquier formato soportado por docling — DOCX, PPTX, XLSX, PDF, HTML, EPUB, imágenes, y más) | `/bedrock:learn` |
| Crear o actualizar entidades manualmente | `/bedrock:preserve` |
| Desduplicar y comprobar la salud del vault | `/bedrock:compress` |
| Re-sincronizar entidades con fuentes externas | `/bedrock:sync` |
